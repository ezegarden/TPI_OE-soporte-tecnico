\# TPI - Organización Empresarial



\## Chatbot de Soporte Técnico Nivel 1



\### Descripción



Este proyecto consiste en el desarrollo de un chatbot para automatizar el proceso de atención de incidencias de Soporte Técnico Nivel 1 dentro de una organización.



El sistema permite:



\* Identificar usuarios mediante legajo o nombre.

\* Clasificar problemas reportados.

\* Consultar una base de conocimientos simulada.

\* Ofrecer soluciones automáticas.

\* Generar tickets de soporte.

\* Escalar casos a Soporte Técnico Nivel 2 cuando sea necesario.



\### Tecnologías utilizadas



\* Python

\* Tkinter

\* BPMN 2.0

\* GitHub



\### Ejecución



1\. Descargar o clonar el repositorio.

2\. Abrir una terminal en la carpeta del proyecto.

3\. Ejecutar:



```bash

python chatbot\_soporte\_tecnico.py

```
